Test your Magento with Menta

Menta is Acceptance Tests Framework for Magento

Download this project as a .zip file Download this project as a tar.gz file

Interested in Selenium tests of your Magento?

We can help you with that. Menta is an acceptance tests framework based on PHP Webdriver, which uses Selenium Server for testing. Is it hard? No - we have created general components for Magento.

Getting started

First thing you need to have is Magento 1.8 working installation. Easiest way to install Magento is in this short tutorial: Magento - Wiki - Installing Magento Via SSH

Next you need to download Menta. We have created sample project to help you with the first setup. Clone our sample project repository:

$ git clone https://github.com/macopedia/Menta_MagentoSampleProject.git

Menta is using Composer. We need to install all required components:

$ ./composer.phar install

Menta can generate HTML reports for each tests. This is a great feature that can be connected with your CI environment. You need to create a directory for them:

$ cd Tests
        mkdir -p ../build/reports

Now you need to run the Selenium Server. This is very easy in Unix OS. Just go to http://docs.seleniumhq.org/download/ and download the latest version of Selenium Server. After downloading you can run Selenium server:

$ java -jar selenium-server-standalone-.jar

Last step of the preparation is Menta setup. You need to create in Magento user for fronted, backend and also for Soap. Then you need to check the setup of Menta in conf/defaults.xml file.

Now it's time for magic. Run this command and you will make your first Menta test:

$ ../bin/phpunit --configuration=../conf/defaults.xml General/ScreenshotsTest.php

If your test runs succesfully it's time for all the tests:

$ ../bin/phpunit --configuration=../conf/defaults.xml
            ../vendor/aoemedia/menta/lib/Menta/Util/CreateTestSuite.php
